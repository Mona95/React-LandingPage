## React - Landing Page
-------------------------------

### How to run the project

- Unzip the folder.
- run `npm install`.
- run `npm start`.

<hr/>

### Used packages 
- fontawsome 
- react-fontawesome
- node-sass
- react-responsive
- uuid
- material-ui/core

<hr/>
